# To-Do-List project

A simple ToDoList project made by me in the process of learning JavaScript. It is not a proper App.
It contains basic HTML structure, required CSS, and JavaScript.

## FEATURES
->Add Task
->Delete Task
->Mark when the task is done
->Three buttons: Active, pending, completed, present to show Tasks
->A counter showing the number of Remaining Tasks.

## Technologies Used
-HTML
-CSS
-JavaScript
